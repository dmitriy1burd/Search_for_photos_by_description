Для фотохостинга разработана демонстрационная версия системы поиска референсных фотографий по текстовому описанию.
Модель на основе векторных представлений изображений и текста определяет степень соответствия между текстовым запросом и изображением, выдавая оценку от 0 до 1

Основные этапы работы:
1. Загрузка и изучение общей информации о данных.
2. Предобработка данных:
  - Агрегация экспертных оценок;
  - Преобразование оценок в вероятностное значение (0-1);
  - Очистка данных от упоминаний детей (юридические ограничения).
3. Подготовка данных для модели:
  - Векторизация текстов с использованием CLIP;
  - Векторизация изображений с использованием CLIP;
  - Разделение на тренировочную и валидационную выборки (90:10).
4. Обучение моделей:
  - Сравнение нескольких моделей: Ridge, Lasso, ElasticNet, LinearRegression;
  - Разработка полносвязной нейросети для вычисления косинусного сходства;
  - Оптимизация гиперпараметров с использованием Optuna.
5. Тестирование лучшей модели:
  - Разработана функция для поиска изображения по текстовому запросу;
  - Реализована фильтрация запросов с упоминанием детей.
